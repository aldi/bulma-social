Social Buttons for Bulma
============================

Social Buttons based on
[Bulma](http://bulma.io/) and
[Font Awesome](http://fortawesome.github.io/Font-Awesome/)

[Check the live demo!](http://aldi.github.io/bulma-social)

Installation
------------

Include the `bulma-social.css` or `bulma-social.min.css` in your project.

Available classes
-----------------
 - `is-adn`
 - `is-bitbucket`
 - `is-dropbox`
 - `is-facebook`
 - `is-flickr`
 - `is-foursquare`
 - `is-github`
 - `is-gitlab`
 - `is-google`
 - `is-instagram`
 - `is-linkedin`
 - `is-microsoft`
 - `is-odnoklassniki`
 - `is-openid`
 - `is-pinterest`
 - `is-reddit`
 - `is-soundcloud`
 - `is-tumblr`
 - `is-twitter`
 - `is-vimeo`
 - `is-vk`
 - `is-yahoo`
 
### Seperate button styles for each brand: Work in progress.

Examples
--------

```html
<!-- Github Button -->
<a class="button is-medium is-github">
    <span class="icon">
      <i class="fab fa-github"></i>
    </span>
    <span>GitHub</span>
</a>

<!-- Github Button Icon -->
<a class="button is-medium is-github">
  <span class="fab fa-github"></span>
</a>
```
